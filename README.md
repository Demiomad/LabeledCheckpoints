<p align="center">This mod adds a <b>percentage label</b> to checkpoints.</p>

# How do I get it?
1. Install [Geode](https://geode-sdk.org/install)
2. Open **Geometry Dash**
3. Click the Geode button
4. Click on "Download"
5. Type "Labeled Checkpoints", then click "View" on the first result.
6. Click "Install" and restart your game!
